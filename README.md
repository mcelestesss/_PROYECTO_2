# Laberinto - Juego POO

Un juego de laberinto con dos modos: ESCAPA y CAZADOR.

## Características
- Modo ESCAPA: Escapa del laberinto mientras los enemigos te persiguen
- Modo CAZADOR: Atrapa a 10 enemigos antes de que escapen
- Sistema de trampas en modo ESCAPA
- Puntuaciones y rankings
- Efectos de sonido y música

## Requisitos
- Python 3.6+
- PyGame 2.5.2

##Juego
-Para jugar selecciona alguno de los dos modos "Cazador" o "Escapa" y seguidamente tendrás que utilizar las flechas de tu teclado para moverte
-En caso de ser cazador puedes usar los túneles (morados) para seguir a tus presas
-En caso de escapar de las demás entidades puedes usar las lianas (amarillos) para escabullirte en el laberinto
-Tú eres el personaje rojo, acumula puntos para ver tu puntuación final y ver tu nombre en el top 5 de los más hábiles, a jugar!
